Technologies Learned

Git


Git is the industry standard in terms of version control software. Although it can be tricky to learn, once mastered it makes collaboration with a team on a project so much easier than before. Git can automatically track changes made by each team member and detect conflicts in order to prevent the whole project from being overwritten every time someone sends their changes to the remote repository. This is in addition to a full suite of other useful functions and a web-based file hosting system called GitHub.
